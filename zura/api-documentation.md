# API Documentation

## API Information

**Type:** BindableEvent\
**Location:** Child of a zone folder\
**Execution method:**

```lua
API:Fire("CommandName", ...) -- additional arguments can be added
```



## API Commands

### PowerOn

Powers on all displays in the zone.

### PowerOff

Powers off all displays in the zone.

### Restart

Restarts all displays in the zone.
